# Colección de Códigos USSD – Cuba

> **Transfermóvil · ETECSA · Telefonía fija y móvil**

Esta guía reúne los códigos USSD más útiles para operar en Cuba desde cualquier teléfono móvil (Android, iOS o dispositivos no inteligentes).  
No necesitas tener saldo de datos ni la aplicación Transfermóvil instalada.

> [!warning]
> Los códigos pueden cambiar por parte de los bancos o ETECSA.  
> Si encuentras errores o faltantes, contribuye en la sección **[Contribuciones](#contribuciones)**.

# 📡 ETECSA / Cubacel – Servicios móviles

Códigos para gestionar la línea móvil (Cubacel), consultar saldo, comprar paquetes, recargar, etc.

## 📊 Consultas de saldo y bonos

| Descripción                                                                  | Código USSD                                      |
| ---------------------------------------------------------------------------- | ------------------------------------------------ |
| Saldo principal, voz, SMS, datos y vigencia                                  | `*222#`                                          |
| Cuánto puedes recargar en el mes (límite 360 CUP) y cuándo puedes la próxima | `*222*732#`                                      |
| Consultar bono de recargas en promoción                                      | `*222*266#` (también para bonos y planes en USD) |
| Consultar paquete de datos comprado (sin bono)                               | `*222*328#`                                      |
| Crédito del plan de voz (si lo compraste)                                    | `*222*869#`                                      |
| Crédito del plan de SMS (si lo compraste)                                    | `*222*767#`                                      |
| Consultar saldo de telefonía fija                                            | `*118#`                                          |
| Saber si tu línea tiene tarifa diferenciada (te llega un SMS)                | `*111*6#`                                        |
| Consulta de MB Cubacel (megas disponibles)                                   | `*222*887#`                                      |
| Consultar el estado del Plan Amigos                                          | `*222*264#`                                      |
| Estado del paquete de bolsa de correo Nauta                                  | `*222*328#`                                      |

## 💰 Recargas de saldo

| Método                                     | Código / Instrucción                                       |
| ------------------------------------------ | ---------------------------------------------------------- |
| Recargar con tarjeta de cupón (voz guiada) | Marcar `*666` y seguir instrucciones                       |
| Recarga rápida con cupón                   | `*662*código de recarga#` (ej. `*662*123456789012#`)       |
| Límite de recarga nacional                 | **360 CUP cada 30 días**                                   |
| Transferencia de saldo entre líneas        | `*234#` (menú) o directamente `*234*1*número*clave*monto#` |
| Adelanto de saldo                          | `*234#` opción 3, o `*222*233#`                            |

## 📦 Planes y bolsas (datos, voz, SMS)

| Acción                                           | Código USSD                |
| ------------------------------------------------ | -------------------------- |
| Comprar servicios (datos, SMS, voz, Plan Amigos) | `*133#` → menú interactivo |
| Consultar estado del plan de voz                 | `*222*869#`                |
| Consultar estado del plan de SMS                 | `*222*767#`                |
| Consultar estado del Plan Amigos                 | `*222*264#`                |

> Para comprar un plan necesitas saldo principal igual o superior al precio del plan.

# 📞 Códigos generales de telefonía

Estos códigos funcionan en casi cualquier teléfono móvil, independientemente del operador en Cuba.

## 🔢 Identificador del equipo (IMEI)

| Descripción                  | Código  |
| ---------------------------- | ------- |
| Mostrar el IMEI del teléfono | `*#06#` |

## 🙈 Llamadas anónimas (ocultar identidad)

| Acción                                                       | Código       |
| ------------------------------------------------------------ | ------------ |
| Ocultar mi número para una llamada (marcar antes del número) | `#31#numero` |
| Desactivar ocultamiento permanente (mostrar ID)              | `*31#`       |

## 🔁 Desvío de llamadas

| Acción                                             | Código        |
| -------------------------------------------------- | ------------- |
| Activar desvío de todas las llamadas a otro número | `*21*número#` |
| Cancelar todos los desvíos                         | `#21#`        |

## 🚫 Restricción de llamadas (bloqueos)

Todos estos códigos usan la clave por defecto `0000` (a menos que la hayas cambiado).

| Acción                                                  | Código       |
| ------------------------------------------------------- | ------------ |
| Cancelar **todos** los bloqueos (entrantes y salientes) | `#330*0000#` |
| Cancelar bloqueo de **llamadas entrantes**              | `#35*0000#`  |
| Cancelar bloqueo de **llamadas salientes**              | `#33*0000#`  |

---

## 📌 Notas importantes

- Todos los códigos USSD se marcan como una **llamada normal** (presiona "llamar" después de escribir el código).
- **Límite de recarga nacional:** 360 CUP cada 30 días (vía Transfermóvil, Servicios En Línea, Monedero, oficinas).  
  Las **transferencias de saldo entre líneas** y las **recargas internacionales** no tienen ese límite.
- Puedes comprar la cantidad de paquetes que desees, siempre que tengas saldo principal suficiente.
- Para comprar un plan (datos, voz, SMS) la línea debe estar activa y con saldo ≥ precio del plan.
- Los códigos de Transfermóvil son específicos para cada banco. Verifica con tu banco antes de operar.
